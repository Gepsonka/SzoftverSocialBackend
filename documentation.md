# Social app

The aim of this app is to provide an easy to use social application.
The user will be able to post pictures and text, post comments.

## Functions:
- User management:
    - User login
    - User registration
    - User deletion
- Ability to create posts:
    - Post pictures
    - Post text
    - Comment under posts
- Edit posts:
    - Delete posts
    - Delete comment
    - Remove picture
-  Editing user:
    - Edit profile picture
    - Edit profile description
    - Edit username
- Post interaction:
    - Liking posts
    - Comment under posts
    - Edit comment
